Two Sum

The Problem

Given a list of numbers and a target number, find two numbers in the list that sum up to the target.

  • Return a tuple/array/set containing the two numbers (e.g. [2, 4]).
  • The index of the returned numbers must be different.
  • If there is no sum, return null or throw an error.

Examples

Here are some test cases you can try out as input:

Input Output
[ 1, 2, 3 ], 5 (2, 3)
[ 5, 4, 6 ], 10 (4, 6)
[ 5, 5, 3 ], 10 (5, 5)
[ 1, 2, 3 ], 10 null or throw an error

Solution

The solution presented in this article has a runtime complexity of O(n) and a space complexity of O(n). Of course, there are many other solutions that are just as good.

This solution loops through the given list of numbers once. The loop maintains a Hash Map that is used to tell whether a certain number exists in the given list of numbers. Note that inserting and accessing elements in a Hash Map only requires O(1) time.

Pseudocode

------------------------------------------
Input: numbers (a list), target (a number)
------------------------------------------

number_hash_map = new hash map

for i = 0, 1, 2, ..., length of numbers - 1
    current_number = numbers[i]
    number_needed = target - current_number
    if number_hash_map is true
        return (current_number, number_needed)
    set value at key current_number of number_hash_map to true

return null (since no pair exists)

JavaScript Solution

function twoSum(numbers, target) {

    var numExists = {};

    for (let i = 0; i < numbers.length; i++) {
        let currNum = numbers[i],
            numNeeded = target - currNum;
        if (numExists[numNeeded]) {
            return [currNum, numNeeded];
        }
        numExists[currNum] = true;
    }

    return null; // No pair found.
}
